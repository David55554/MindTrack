# Journal API

Author: David Okyere (amo1do@alma.edu)

Environment variables (create a .env file):
```
OPENAI_API_KEY=sk-...
FIREBASE_PROJECT_ID=...
FIREBASE_CLIENT_EMAIL=...
FIREBASE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n....\n-----END PRIVATE KEY-----\n"
PORT=5050
```

Run:
```
cd journal-api
npm install
npm run dev
```
